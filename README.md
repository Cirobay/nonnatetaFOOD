the AdminLTE control panel,
Livewire-Jet Stream

## Usage

-   **composer install**

-   **mv .env.example .env**

-   **php artisan cache:clear**

-   **composer dump-autoload**

-   **npm install**

-   **npm run dev**

-   **php artisan key:generate**

-   **php artisan migrate:fresh --seed**
